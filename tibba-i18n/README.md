# tibba-i18n

**错误消息 i18n**

> **分层**：标准（Standard）— 标准 REST 构件，依赖核心

按 Accept-Language 协商语言并本地化错误响应 message。

## 依赖

依赖：tibba-error

## 在工作区中的位置

- 版本：与 workspace 统一（`version.workspace = true`，当前见根 `Cargo.toml` `[workspace.package]`）
- 发布：见 `scripts/publish.sh`（`core` / `ext` 分组）
